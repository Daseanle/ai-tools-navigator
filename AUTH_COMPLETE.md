## 🌐 **如何使用**

### 🚀 **访问网站**
```
主页: http://localhost:3000
中文版: http://localhost:3000/zh
```

### 👤 **用户操作流程**

1. **注册新用户**
   - 点击右上角"登录"按钮
   - 切换到"注册"标签
   - 填写邮箱和密码（至少6位）
   - 点击"注册"按钮
   - 检查邮箱验证邮件

2. **登录现有用户**
   - 点击右上角"登录"按钮
   - 在"登录"标签中填写邮箱密码
   - 点击"登录"按钮
   - 登录成功后显示用户头像

3. **用户菜单功能**
   - 点击用户头像查看菜单
   - 查看"我的收藏"
   - 查看"我的评价"
   - 访问"设置"
   - "退出登录"

## 💡 **Supabase控制台配置建议**

1. **启用邮箱确认**
   - Authentication → Settings → Enable email confirmations

2. **配置重定向URL**
   - Authentication → URL Configuration
   - Site URL: `http://localhost:3000`
   - Redirect URLs: `http://localhost:3000/auth/callback`

3. **自定义邮件模板**
   - Authentication → Email Templates
   - 自定义确认邮件和重置密码邮件
